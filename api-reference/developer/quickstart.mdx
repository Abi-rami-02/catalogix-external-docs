**Step 4: Define Store Taxonomy**
The user can choose to define the taxonomy from scratch or use the Streamoid MarketPlace (SMP) directly.
If the user defines the taxonmy from scratch then it there needs to be a one-time mapping of the taxonomy to SMP.<br>

**Step 5: Import Products.**
The user can import products to their catalog store through multiple ways: CSV import, feed URL import, Shopify store connector. More a detailed guide on importing products to a store.
Refer: (hyperlink to import products section).<br>

**Step 6: Run an automation.**
Catalogix provides a large number of automations like AI-powered attribution, AI powered product content generator, etc that can be used for speeding up the cataloging process. After the products have been imported to the store. You can select your products run the desired automatin on them. You can track the progress/completion of the automation from the Activity tab.<br>

**Step 7: Export Products**
After the automation has been run on the products and the desired results have been achieved then you can export the products from the store in any of required format. You can export the products in the current view( the exported file will have data in same format as  the PLP view) or in the template of the marketplace you want to list on.
