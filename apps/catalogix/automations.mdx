### Overview

Catalogix automations combine AI vision and generative capabilities to handle repetitive catalog work at scale: image attribution, marketplace translations, content generation, and more. 

### Run an Automation

<Steps>
    <Step title="Select products">
      Choose the products you want to run the automation on. You can also select the entire store by clicking the dropdown. ![Select products for group edit](https://assets.catalogix.ai/b2088be9ba781ffd4d209d858f881106) ![Select store dropdown](https://assets.catalogix.ai/9d74d8d2a58957c3b577b2dc1d5b4a26)
    </Step>
    <Step title="Click the Automation button">
      Click **Automation** to open the automation menu. ![Group edit toggle](https://assets.catalogix.ai/3411ea1ee6cbc7b329e965c2fb0318b9)
    </Step>
    <Step title="Browse and select the required automation">
     Review the automation details, configure any required settings, and click **Run automation**. ![Bulk edit attribute](https://assets.catalogix.ai/0d2ede344144d8d2b4b06e049afae99a)
    </Step>
    <Step title="Check Status">
     After the automation starts, monitor progress in the **Store Activity** tab. ![Bulk edit attribute](https://assets.catalogix.ai/bc6033495987635f15ac4bb8b023e8ff)
    </Step>
  </Steps>

<Tip>
Every automation uses credits. Review your available credits before running an automation. 
</Tip>

### Automation List Overview

Automations are split into two groups: Basics and Enhancers. 

### Basic Automations

### Generate Attributes from Images (Express & Precise)

This automation analyzes product imagery to populate attributes like color, pattern, or category. Run it against selected attributes when you need targeted attribution.

![Generate attributes from images configuration](https://assets.catalogix.ai/d630cd00be55903d83ccc4ba4517a3fe)

![Image attribute generation results](https://assets.catalogix.ai/bc7e10ffd0a46d19be15cc03959b215c)

Attribute labels indicate their origin:

- **User Data**: manually edited values  
- **Feed Data**: imported via feed (i.e. Excel/CSV)  
- **Image Data**: generated by automations  

Catalogix applies values using the precedence **User > Feed > Data**. Select **Override existing data** if automation output should replace the current value.

### Identify Image Type

This automation identifies and labels the **pose type of each product image (e.g. front, back, side)**. You can run this automation without making any selections in the menu. 

![Identify image type](https://assets.catalogix.ai/fa170fa926c8d385e73262ec0e324a33)

Once the automation is complete, you can check/QC the results by **opening the PDP and navigating to the multimedia section** 

![Identify image type](https://assets.catalogix.ai/f497555df79fbc73802ed2a4b2a40297)

![Identify image type](https://assets.catalogix.ai/ebd73d9b65c74aed29bd27e56f6fe409)

Change the pose type of an image if you think the prediction is incorrect or you want to change it. Simply click on the dropdown and make an appropriate selection. 


### Remove Background from Images

This automation removes the background of product images and prepares it for marketplace translations. Select the 'exclude props' option if you want to also remove props from the background of images. 

<Tip>
If you want to use your own background for marketplace listings, you don't need to run this automation.   
</Tip>

![Remove BG from images](https://assets.catalogix.ai/122af9df45666acb756804efdc02d6ae)

### Translate Attributes for Marketplaces

This automation translates or converts product data from the base taxonomy to the target marketplace taxonomy. It also ensures that the converted data is filled in the correct template. 

- **Source Marketplace**: marketplace that supplies reference data (i.e. SMP)
- **Target Marketplace**: marketplaces that the data should be converted to (i.e. Amazon, Flipkart, Myntra, etc.) 
- **ALLOWED_CONFIG_OPTIONS**: use the value 'default' here 

![MP translations](https://assets.catalogix.ai/bf0614df1aa0b36b3b383fd4cff2ed29)

<Tip>
Select **Override existing data** if automation output should replace the current value.  
</Tip>

### Translate Images for Marketplaces

This automation translates or converts product images to the required target marketplace dimensions, background, file size, etc. It also allows minor customizations for backgrounds and sequencing.  

![MP translations](https://assets.catalogix.ai/98d91608f5cb1109467541f11a62d62a)

- **Source Marketplace**: marketplace that supplies reference/original images (i.e. SMP)
- **Target Marketplace**: marketplaces that the images should be converted to (i.e. Amazon, Flipkart, Myntra, etc.) 
- **Translation options**: select an option based on your use case 

![MP translations](https://assets.catalogix.ai/3e71a7f72843aff7e774cb18c7d41a9c)

### Translation Options Overview
- **use_marketplace_bg_and_dimensions** - All images will be converted to match the target marketplace background and dimension requirements. 
- **use_marketplace_dimensions_with_custom_bg** - All images will be converted to match the target marketplace dimensions but the background color will be custom. 
- **keep_original_bg_and_dimensions** - No changes will be made to the images 
- **keep_bg_and_convert_to_marketplace_dimensions** - Background will be retained as original and only the image dimensions will be converted as per the target marketplace requirements.
- **convert_bg_and_dimension_of_first_image_only** - Background and dimensions will be converted as per target marketplace requirements only for the first image. 
- **keep_original_image_urls** - No changes will be made to the images and the image links will remain the same as provided in the CSV. 
 
<Tip>
Select **Override existing data** if automation output should replace the current value.  
</Tip>

### Enhancer Automations

### Analyze Image or Text

Use AI vision plus a custom prompt to generate marketplace-ready content from a selected product image.

- **Type**: Select whether your input is an image or text  
- **Source Marketplace**: marketplace that supplies reference data  
- **Source Image Attribute**: product image used as input (shown only if type=image)  
- **Source Text Attribute**: product attribute used as input (shown only if type=text)  
- **Target Marketplace**: marketplace that receives the output  
- **Target Attribute**: attribute populated with generated content  
- **Prompt Message**: instructions that shape the AI output  

![Analyze image with prompt form](https://assets.catalogix.ai/302df3908873749ba0bfd09d6c6b8bfd)
