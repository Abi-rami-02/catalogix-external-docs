## Overview

Store Settings centralizes every configuration area for a Catalogix store. Use these modules to adjust defaults, connect feeds, sync exports, and curate taxonomy for higher-quality automation.

## General

- Manage **Store Details** such as name, region, business type, and active channels.  
- Review **Channel Versions** to confirm listings are in sync.  

![General Settings](https://assets.catalogix.ai/185711f2108d751f485378f77d748b42)

## Default Automation

Choose which automations run automatically the moment products enter the store so automation starts without manual triggers.

![Automation Settings](https://assets.catalogix.ai/77148165f241d5f57def7cd87607c4f5)

## Import

Monitor incoming feeds, review import history, and troubleshoot ingestion issues across the entire store.

![Import Settings](https://assets.catalogix.ai/711dbd6dff8af5b9e3944fef33eae06c)

## Export

Track external syncs, destination templates, and export history for the entire store. 

![Export Settings](https://assets.catalogix.ai/51666d003c5b97b8b0c8d285dd375bef)

## Curation

Refine Catalogix taxonomy by selecting relevant attributes and values to improve AI accuracy.
