## Overview

The Product Listing Page (PLP) shows a consolidated list of products in a Catalogix store. Switch views to focus on the information you need, edit product details, run automations, and monitor store activity from a single screen.


## Switch to pre-loaded views

Once you have uploaded products to your store you can now start checking for data quality. To do so, you can switch your views based on category. 

<Info>
A view is a specifc collection of columns/attributes that are actively visible to you on the PLP.
</Info>

<Steps>
  <Step title="Click on on the 'Default' icon on the top-right">
  ![Create Store](https://assets.catalogix.ai/fcb80f7da510c440ff3ebaf1b324acfb)
  </Step>
  <Step title="Choose Template/View">
  Choose from the list of available views. You can search for a category as well and click on the one you want. ![Store Details](https://assets.catalogix.ai/d02cad18c9c0de1bedd1b670dc948b72)
  </Step>
  <Step title="Changed view with Category filter">
  Once you have selected your view, you can now see that your PLP is changed with a category filter and relevant attributes. This helps you focus on reviewing the data for that category. ![Channels](https://assets.catalogix.ai/71e282bd346320b3dd0552beafb9f25a)
  </Step>
</Steps>

## Create your own views 

<Info>
You can create your own views i.e collection of columns/attributes instead of using pre-loaded views.
</Info>

<Steps>
  <Step title="Click on on the 'Manage columns' icon on the top-right">
  ![Create Store](https://assets.catalogix.ai/edcc26462a62f949ba3444d3cd8ccefd)
  </Step>
  <Step title="Add/Delete columns">
  All the visbile columns are on the right & the other hidden columns are on the left. Select attributes from either side and use the hide/make visbile button to create your view. ![Store Details](https://assets.catalogix.ai/1b869918af82b78a873b61a939baf480) 
  </Step>
  <Step title="Re-order columns">
  You can set your required column order by clicking on the hold icon next to each attribute and dragging it up or down  ![Channels](https://assets.catalogix.ai/3f82506c1b0cdd0356f7f0025fcf1645)
  </Step>
  <Step title="Lock Columns">
  You can lock the column i.e freeze it by turning on the toggle. Please note that at any given time only 3 columns can be locked. ![Channels](https://assets.catalogix.ai/c159a6a9980bcf1c039440f204c8dee4)
  </Step>
  <Step title="Exit and Save the view">
  Click outside the modal and then save the view. You can give the view any name you prefer. ![Channels](https://assets.catalogix.ai/7cd70d4d635d45efcf9452b2205ff5d2) ![Channels](https://assets.catalogix.ai/aed961e1702e822ba6ed2851b157686c)
  </Step>
</Steps>

All your created views will be present under the 'views' tab in the change veiws section. 

![Views](https://assets.catalogix.ai/9866bf0b3a2a5dddede871183e9e637f)

<Info>
You can create multiple views based on your use case. Views can also be combined with filters and saved. 
</Info>

## Bulk Edit

Bulk edit lets you update a attribute across multiple products at once. This can be done even across the entire store. 

  <Steps>
    <Step title="Select products">
      Choose the products you want to edit together. ![Select products for group edit](https://assets.catalogix.ai/b2088be9ba781ffd4d209d858f881106)
    </Step>
    <Step title="Turn on bulk edit">
      Toggle **Group edit** to activate bulk editing mode. ![Group edit toggle](https://assets.catalogix.ai/856f83951afee606ec62a6325ccae286)
    </Step>
    <Step title="Apply the change">
      Update the attribute once and Catalogix applies it to every selected product. ![Bulk edit attribute](https://assets.catalogix.ai/75a55a3cbbba74f74d2867003354e9b6)
    </Step>
  </Steps>

## Data Highlights

Use highlights to quickly spot data gaps, errors, corrections, duplicates, and character counts across the PLP.

## Enable Highlights

Open the **Highlight** menu in the PLP to toggle any of the options above.

![Highlight menu default](https://assets.catalogix.ai/1864490740fd841218f3f1a2473692dc)

![Highlighted rows](https://assets.catalogix.ai/dada85514fdc12a50d7cee0ebd74c758)

### Data Gaps
Marks attribute values present in images (via automations) but missing from the feed. No legend appears if the classifier has not run or the feed value already exists.

- *Green*: classifier has high confidence
- *Orange*: classifier has medium confidence
- *Red*: classifier has low confidence
- *Black*: no confidence score available

### Data Errors
Shows where automation output differs from feed data. Legends display only when the feed contains a value.

Value precedence on the PLP: **User &gt; Feed &gt; Automation** (active values can be user or feed sourced).

- *Green*: active and automation values match with high confidence
- *Orange*: active and automation values match with medium confidence
- *Red*: active and automation values match with low confidence, or the values differ
- *Black*: no confidence score available

### Data Corrections
Highlights values that users changed from their feed or automation source.

- *Green*: user and automation values match
- *Orange*: user and automation values differ
- *Red*: feed and automation values match, but the user entered a different value
- *Black*: no confidence score available (no automation value present)

### Character Count
Monitors the length of free-text attributes such as product names or descriptions.

![Character count indicator](https://assets.catalogix.ai/847a52e3aeb002ef48f7437fa477fb8b)
